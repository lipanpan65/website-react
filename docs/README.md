feat：新功能（ feature ）
fix：问题修复
docs：文档
style：调整格式（不影响代码运行）
refactor：重构
test：增加测试
chore：构建过程或辅助工具的变动
revert：撤销以前的提交
scope 用于说明提交的影响范围，内容根据具体项目而定。
subject 用于概括提交内容。
